Exportar parada

Página principal > Inicio de sesión > Menú parada > Exportar parada

En esta ventana puedes exportar las estancias de una parada dentro de un rango de fechas específico. Además, se muestran los datos principales de la parada, como el ID, el responsable, el nombre, la región y el correo electrónico. Selecciona el período deseado, visualiza la información de las estancias en pantalla y exporta los datos si es necesario.

Este botón abre la ayuda de la aplicación.

En este campo debes seleccionar la fecha de inicio del rango de fechas que deseas consultar o exportar.

En este campo debes seleccionar la fecha de fin del rango de fechas que deseas consultar o exportar.

Utiliza este botón para buscar y visualizar las estancias correspondientes al rango de fechas seleccionado en tu parada.

Utiliza este botón para generar un informe detallado de las estancias realizadas en el rango de fechas seleccionado. El informe será exportado en formato pdf.

Este botón te permite volver a la ventana anterior. Si tienes dudas, consulta la ruta de navegación en la parte superior de esta sección de ayuda.

Este botón te permite salir de la aplicación.
