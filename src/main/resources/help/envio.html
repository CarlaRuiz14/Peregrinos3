Envío a casa

Página principal > Inicio de sesión > Menú parada > Sellar parada > Alojamiento en parada > Contratar Servicios > Envío a casa

En esta ventana puedes gestionar el envío de paquetes a domicilio. Introduce la dirección y localidad de destino, así como las características del paquete, incluyendo dimensiones, peso y si el envío es urgente.

Este botón abre la ayuda de la aplicación.

En este campo se introduce la dirección a la que será enviada el paquete. Esta puede contener letras, números y los caracteres: ".", ",", "/","ª","º" y "-".

En este campo se introduce la localidad a la que será enviada el paquete. Esta puede contener letras y "-".

Para contratar un envío urgente esta opción debe estar marcada.

En este campo se introduce el peso en kilogramos del paquete a enviar. Puede ser un número decimal o entero.

En este campo se introduce el alto en centímetros del paquete. Puede ser un número decimal o entero.

En este campo se introduce el ancho en centímetros del paquete. Puede ser un número decimal o entero.

En este campo se introduce el fondo en centímetros del paquete. Puede ser un número decimal o entero.

Haz clic en este botón para confirmar los datos y contratar el envío a casa con los datos introducidos.

Este botón te permite volver a la ventana anterior. Si tienes dudas, consulta la ruta de navegación en la parte superior de esta sección de ayuda.

Este botón te permite salir de la aplicación.
