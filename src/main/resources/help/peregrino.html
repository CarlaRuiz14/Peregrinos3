Página principal > Inicio de sesión > Menú peregrino

Bienvenido al menú de peregrino. Desde esta sección puedes realizar diversas acciones, como exportar tu carnet de peregrino, editar tu información personal o cerrar sesión. Utiliza los botones disponibles para navegar y gestionar tus datos de manera sencilla.

Este botón abre la ayuda de la aplicación.

Haz clic en este botón para exportar el carnet de peregrino. Podrás visualizar la información del carnet y guardarla en un archivo pdf para su uso o impresión.

Haz clic en este botón para editar la información del usuario. Podrás modificar datos como el correo electrónico, apellidos y otra información personal según sea necesario.

Este botón te permite cerrar la sesión y volver a la pantalla de inicio de sesión.

Este botón te permite salir de la aplicación.
