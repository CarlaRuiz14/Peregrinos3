Editar usuario

Página principal > Inicio de sesión > Menú peregrino > Editar usuario

En esta ventana puedes editar la información de tu perfil de usuario. Aquí podrás actualizar tus datos personales. Asegúrate de guardar los cambios una vez que hayas realizado las modificaciones necesarias.

Este botón abre la ayuda de la aplicación.

Introduce tu nombre. Asegúrate de escribir únicamente letras, sin incluir números ni caracteres especiales, para garantizar la validez del registro.

Introduce tus apellidos. Usa solo letras y evita números o caracteres especiales para cumplir con los requisitos del registro.

Introduce tu email. Este será utilizado para comunicarnos contigo y como método para la recuperación de tu contraseña. Asegúrate de que sea un formato válido (por ejemplo: usuario@dominio.com).

Selecciona tu nacionalidad de entre las opciones disponibles en la lista desplegable.

Haz clic en este botón para limpiar todos los campos del formulario y empezar de nuevo. Esto eliminará cualquier información que hayas introducido.

Haz clic en este botón para completar tu registro como peregrino una vez que hayas ingresado toda tu información y confirmado los datos.

Este botón te permite volver a la ventana anterior. Si tienes dudas, consulta la ruta de navegación en la parte superior de esta sección de ayuda.

Este botón te permite salir de la aplicación.
